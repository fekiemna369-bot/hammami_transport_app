Hammami Transport App : 

A Flutter mobile app designed to make everyday transport management simple and stress‑free.
With Firebase powering the backend, the app helps users stay connected, track their wallet balance, manage expenses, and receive real‑time notifications — all in one place.

✨ Features :

🔐 Secure login & authentication with Firebase

💳 Wallet management: balance, spending limits, monthly expenses

🚫 Smart blocking system: prevents overspending with clear alerts

🔔 Push notifications via FCM for updates and reminders

🖼️ Profile management: photo, contact info, personalized card number

📊 Transaction tracking: see where your money goes each month

🚀 Tech Stack :

Flutter for cross‑platform UI

Firebase Auth & Firestore for authentication and data storage

FCM for notifications
